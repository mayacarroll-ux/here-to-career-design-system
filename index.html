Here to Career Design System

Design tokens and components extracted from the Here to Career platform (career.ja.org). Built on a customized Bootstrap 4 theme: Montserrat typography, a deep-teal palette, pill-shaped actions, and soft-radius cards. Markup mirrors Bootstrap 4 class names so styles drop onto the existing app unchanged.

Get started

The palette centers on deep teal for primary actions, with bright teal and JA blue as accents. Warning is a pale-yellow surface color and always pairs with dark text.

Typography

Montserrat for everything, falling back to Open Sans then system sans. Headings are bold (700) in Primary Dark; body text is regular (400) in Dark.

Buttons

All buttons are pill-shaped with semibold labels. Primary for the main action on a screen; outline for secondary actions alongside it.

<button class="btn btn-primary">Primary</button>
<button class="btn btn-outline-primary">Outline</button>
<button class="btn btn-primary btn-lg">Large</button>

Badges

Pill chips for statuses and category tags. The teal-tint badge is the convention for skill and interest tags.

Getting started

The system is two plain CSS files with no build step and no framework dependency.

<link rel="stylesheet" href="tokens.css">  <!-- design tokens only -->
<link rel="stylesheet" href="h2c.css">     <!-- components (needs tokens) -->
<link href="https://fonts.googleapis.com/css2?family=Montserrat:wght@400;500;600;700;800&display=swap" rel="stylesheet">

<body class="h2c"> <!-- scope base typography -->

To theme an existing Bootstrap 4 project instead, load tokens.css and map Bootstrap's variables to the --h2c-* tokens, or load h2c.css after Bootstrap — class names match.
